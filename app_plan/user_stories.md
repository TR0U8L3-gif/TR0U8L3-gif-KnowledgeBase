# User Stories

This document translates the use cases into user stories for development.

## MVP

### Navigation & Reading

*   **As a user, I want to see a list of all available articles and directories in a navigation panel so that I can easily browse the knowledge base.**
*   **As a user, I want to be able to click on an article in the navigation panel and see its content displayed in a central panel so that I can read it.**
*   **As a user, I want the articles and directories to be displayed in a predefined order so that I can follow a logical structure.**
*   **As a user, I want to see a table of contents for the current article so that I can quickly jump to different sections.**

### Theme

*   **As a user, I want to be able to switch between a light and dark theme so that I can choose the most comfortable viewing experience.**

### Search

*   **As a user, I want a search bar to find articles by keywords so that I can quickly find the information I need.**

## Post-MVP / Future Enhancements

*   **As a user, I want the application to remember the last article I was reading so that I can easily pick up where I left off.**
*   **As a user, I want to see the search keywords highlighted in the article text so that I can quickly locate the relevant information.**
*   **As a content creator, I want a visual editor to reorder articles and directories instead of manually editing `.order` files.**
*   **As a user, I want to be able to resize the three panels to my liking.**
